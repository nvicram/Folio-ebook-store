# Folio E-Book Store

A modern serverless-inspired eBook store frontend hosted on AWS S3.

## Features

- Dynamic book rendering
- Categories
- Reviews section
- Shopping cart
- Login / Signup modal
- Responsive UI
- AWS S3 Static Hosting

## Tech Stack

- HTML
- CSS
- JavaScript
- AWS S3
- GitHub

## Future Improvements

- AWS Lambda APIs
- DynamoDB database
- API Gateway
- CloudFront CDN
- User Authentication
